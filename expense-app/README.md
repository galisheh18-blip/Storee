# Мои траты — приложение для учёта расходов

Простое приложение для анализа личных трат: расходы по категориям (числа и таблица),
лимиты и остаток бюджета, плановые платежи (подписки, ЖКХ) со статусом «оплачено / не оплачено».

Работает **офлайн**, все данные хранятся **локально на устройстве** (в браузере), без сервера и регистрации.

## Возможности

- 📊 **Обзор** — сколько потрачено, бюджет и сколько осталось; таблица по категориям с остатком и прогресс-барами.
- ➕ **Траты** — добавление расхода в категорию с комментарием, история за месяц.
- 📆 **Планы** — подписки, ЖКХ, кредиты; переключатель «оплачено / не оплачено» за текущий месяц.
- 🗂️ **Категории** — свои категории и месячные лимиты, редактируются на лету.
- 📅 Переключение месяца (кнопка календаря вверху справа).
- 🌗 Автоматическая тёмная тема.

## Как установить на телефон (PWA)

Это PWA — веб-приложение, которое ставится на телефон «как приложение», без магазина.

1. Разместите папку `expense-app/` на любом HTTPS-хостинге
   (GitHub Pages, Netlify, Vercel — всё бесплатно) или откройте локально (см. ниже).
2. Откройте адрес в браузере телефона:
   - **Android (Chrome):** меню ⋮ → «Установить приложение» / «Добавить на главный экран».
   - **iPhone (Safari):** «Поделиться» → «На экран Домой».
3. На главном экране появится иконка «Траты». Приложение запускается в полный экран и работает офлайн.

> Для установки как PWA сайт должен открываться по **https** (или `localhost`).
> GitHub Pages даёт https автоматически.

## Быстрый запуск локально

```bash
cd expense-app
python3 -m http.server 8000
# затем откройте http://localhost:8000 в браузере
```

## Публикация на GitHub Pages (пример)

1. Запушьте репозиторий на GitHub.
2. Settings → Pages → Source: ветка с этим кодом, папка `/expense-app` (или переместите файлы в корень).
3. Через минуту приложение будет доступно по адресу `https://<логин>.github.io/<репозиторий>/`.

## Структура

```
expense-app/
├── index.html            # разметка и экраны
├── styles.css            # оформление (светлая/тёмная тема)
├── app.js                # вся логика + хранение в localStorage
├── manifest.webmanifest  # манифест PWA (иконки, имя, цвета)
├── sw.js                 # service worker — офлайн-кэш
└── icons/                # иконки приложения (192, 512, maskable)
```
