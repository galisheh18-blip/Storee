# Калории и БЖУ

Оффлайн-приложение (PWA) для подсчёта калорий и БЖУ — аналог MyFitnessPal / FatSecret / Yazio для личного использования. Работает без интернета, все данные хранятся только на устройстве (`localStorage`).

## Возможности

**Учёт питания**
- Дневник по дням (навигация вперёд/назад, выбор даты)
- Приёмы пищи: завтрак, обед, ужин, перекус
- Расчёт калорийности и БЖУ по каждому продукту и итог за день
- Учёт белков, жиров, углеводов в граммах и процентах
- Встроенная база из ~55 продуктов (значения на 100 г)
- Поиск по названию и штрих-коду
- Добавление своих продуктов (в т.ч. со штрих-кодом)
- Редактирование и удаление продуктов
- Рецепты: соберите блюдо из продуктов — приложение посчитает КБЖУ на порцию
- Избранное и «недавние» для быстрого добавления

**Цели и нормы**
- Автоматический расчёт нормы калорий по формуле Миффлина — Сан Жеора
- Учёт пола, возраста, роста, веса и уровня активности
- Цели: похудение (−15%), поддержание, набор массы (+10%)
- Показ BMR и TDEE
- Ручная норма калорий (переключатель)
- Настройка распределения БЖУ в процентах → перевод в граммы
- Остаток/перебор калорий с учётом активности

**Активность и вода**
- Учёт выпитой воды (+/−250 мл, цель по воде)
- Учёт тренировок и сожжённых калорий (готовые пресеты + вручную)

**Аналитика**
- Кольцо прогресса калорий и полоски БЖУ за день
- График калорий за 7 дней с линией цели
- Круговая диаграмма БЖУ за день
- График динамики веса
- Журнал веса и замеров (талия, % жира)
- Счётчик дней подряд (streak)

**Данные**
- Экспорт и импорт (файл `.json` или текст)
- Полный сброс
- Оффлайн-режим через Service Worker, установка «на экран» (PWA)

## Как запустить

Откройте `index.html` в браузере или разместите папку на любом статическом хостинге (например GitHub Pages) и добавьте на главный экран телефона.

## Чего нет (требует внешних сервисов/сервера)

- Распознавание еды по фото и сканирование штрих-кода камерой — требуют онлайн-API и ML, несовместимо с полностью автономным локальным приложением. Штрих-код можно вводить вручную для быстрого поиска.
- Push-напоминания — требуют серверной части.
